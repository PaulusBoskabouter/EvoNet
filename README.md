# EvoNet

Welcome to the GitHub repository for the Natural Computing 2026 project.

## Repository Structure

- `ReLU_experiments.ipynb`: Contains the theoretical (thought) experiments.
- `baseline_experiments.py`: Includes baseline model training, EvoNet model performance, and comparisons.
- `evonet_experiments.py`: Contains the training loop for the EvoNet experiments as described in the report.

## Setup and Dependencies

Use the following commands to set up and run the project:

- To download dependencies:
  ```bash
  uv sync
- To run a file:
  ```bash
    uv run <file>.py
- For the notebooks:
  add the .venv created by uv as interpreter in vsc
